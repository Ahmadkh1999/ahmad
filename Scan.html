Scanning

Ping

Ping is a basic network tool that checks the connection between devices. It sends a small packet to a specific address and measures the time for a response, helping to detect network issues and assess performance.

1- Open Command Prompt (Windows) or Terminal (Mac/Linux). 
2- Type: ping [IP address or domain]. 
3- Press Enter. Review the displayed round-trip time and packet loss. 
4-To stop, use Ctrl + C (Cmd + C on Mac).

tracert

Tracert (traceroute) is a network diagnostic tool used to trace the path that data packets take from your computer to a specified destination, like a website or server.

1- Open Command Prompt (Windows) or Terminal (Mac/Linux).
2- Type "tracert" followed by the target website or IP address.
3- Press Enter to execute the command.
4- View the list of network hops and response times displayed.

nslookup

Nslookup stands for "Name Server Lookup." It is a command-line tool used to query the Domain Name System (DNS) to obtain domain name or IP address information. Nslookup allows users to directly query DNS servers to retrieve various types of DNS records. 

1- Open Command Prompt (Windows) or Terminal (Mac/Linux).
2- Type "nslookup" followed by the domain name or IP address.
3- Press Enter to execute the command.
4- it will display information about the queried domain, including its IP address and associated details like the authoritative DNS server.
